###查找替换

- **在所有手机号码前后增加引号**
```
:%s/\(\d\{11}\)/'\1'/g
```
- **在所有行最前面增加-**
```
:%s/^/-/g
```
- **在所有行的最后增加-**
```
:%s/$/-/g
```
